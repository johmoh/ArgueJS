ANYTYPE Class

Defined in: argue2.js:353
Module: module-arguejs

Constructor of class ANYTYPE. That type is used to mark a parameter as "any type execpt undefined and null allowed".

ANYTYPE is a special type for marking an parameter type as "any type". That means any type is allowed for that parameter - but of course, "undefined" and "null" are not allowed. If you want to specify an parameter that accepts values of any type and "undefined" and "null" you can do that. In such a case speficy "allowUndefind: true" and "allowNull: true" in the specification of the parameter.
